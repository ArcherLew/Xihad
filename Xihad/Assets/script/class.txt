### 通用 SRPG 逻辑
+ 地图模块
	+ 支持多种形状的棋盘（四边形、六边形等等）
	+ 寻路模块
	+ 地图元素
		+ 角色是否可通过
		+ 角色是否可停留
		+ 是否可以受道具、法术影响：地形可能需要知道其上是否有其他地图元素才能知道是否可以受道具影响
		+ 寻路时消耗角色的 AP 值
		+ query()
	+ 地形、角色、障碍物、宝箱等都是地图元素的子类
	+ 地图块上有至少一个地图元素
	+ 地图块一定有且仅有一个地形作为其上的地图元素
+ 角色相关
	+ 角色拥有当前能力值集合、状态集合；角色拥有的技能、角色团队等由子类支持
	+ 能力表达式
	+ 角色状态
		+ 标签：异常状态、超常状态
		+ 类型：中毒、加攻……
		+ 附加时处理：改变状态、附加精灵（可有可无，装备增益无精灵）
		+ 失效时处理：恢复状态、分离精灵
		+ 可能需要监听回合更新：更新剩余回合、中毒时减少血量
		+ prototype copy
	+ 阵形、技能、道具、基础状态都可以给角色附加能力影响状态
	+ 状态可以影响角色能力值，也可以影响角色其他属性
+ 阵形需要监听角色的位置变化
+ 技能/道具相关
	+ 名称
	+ 标签：xx属性、只能对敌人？
	+ 附属状态
	+ 可攻击目标
		+ getNextTargetTile(skillSource, map, prevTile)
	+ 攻击影响范围
		+ getNextAffectableTile(skillSource, map, targetTile, prevTile)
	+ resolve 技能源、技能目标元素
