###文件夹自述

解压后得到的文件夹是一个Tiled的工作空间，包含了Tiled需要的图块文件和导出的tmx文件，lua文件。

###如何导出地图

1.  使用Tiled软件编辑地图，并且将地图导出为lua格式
2.  使用2battle.lua文件处理生成的lua格式的地图文件，指令形如2battle.lua xxx.lua。
